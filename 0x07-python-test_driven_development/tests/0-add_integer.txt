=======================
|  Test Add Integers  |
=======================

this test was made to check if the program to add 
2 integer numbers work correctly.

Sum:
====

``0-add_integer()`` returns the sum of its arguments.

::
#	>>> from '0-add_integer' import add_integer

	100

It also works with floating-point values.

::
	>>> add_integer(100.3, -2)
	98

The program raised errors when are necessary.

::
	>>> add_integer(4, "School")
	Traceback (most recent call last):
	TypeError: b must be an integer

	>>> add_integer("Holberton", 100)
	Traceback (most recent call last):
	TypeError: a must be an integer

	>>> add_integer("Holberton", "School")
	Traceback (most recent call last):
	TypeError: a must be an integer

	>>> add_integer(["Hola"], ["mundo"])
	Traceback (most recent call last):
	TypeError: a must be an integer

	>>> add_integer(float("inf"))
	89

	>>> add_integer(float("NaN"), 0)
	89
